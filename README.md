# Password Generator

A simple and efficient web-based tool to generate strong, random passwords. This project is built using HTML, CSS, and vanilla JavaScript.

## Features

-   **Random Password Generation**: Generates a 9-character password including a mix of:
    -   Lowercase letters
    -   Uppercase letters
    -   Numbers
    -   Special characters
-   **One-Click Copy**: Easily copy the generated password to your clipboard with a single click.
-   **Responsive Design**: Clean and user-friendly interface with a dark theme.
-   **Visual Feedback**: Notifies the user when the password has been copied.

## Technologies Used

-   **HTML5**: For the structure of the application.
-   **CSS3**: For styling, including Google Fonts, flexbox layout, and hover effects.
-   **JavaScript**: For the password generation logic and DOM manipulation.

## How to Use

1.  Clone the repository or download the `pass.html` file.
2.  Open `pass.html` in any modern web browser.
3.  Click the **"Generate a password"** button to create a new password.
4.  Click the **"Copy"** button to copy the password to your clipboard.

## Project Structure

-   `pass.html`: Contains the HTML structure, CSS styling, and JavaScript logic in a single file.

## Setup

No installation is required. Simply open the file in a browser to run the application.

```bash
# Clone the repository (if applicable)
```
